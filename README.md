# mortgage-calculator

等额本息固定利率提前还房贷计算器。

## 运行效果

![提前还房贷计算器运行效果](docs/assets/screenshot.png)

## 功能

- 支持按月供、剩余期数和利率反推当前剩余本金
- 支持第 N 期后提前还款
- 支持直接填写提前还款金额
- 支持按“提前还多少期”折算对应本金
- 对比“月供不变，缩短期限”和“期限不变，降低月供”
- 显示节省利息、净节省和关键时间线

## 本地运行

```bash
npm start
```

然后打开：

```text
http://127.0.0.1:4173/
```
