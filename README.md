# mortgage-calculator

等额本息固定利率提前还房贷计算器。

## 功能

- 支持填写当前剩余本金，或按月供、剩余期数和利率反推本金
- 支持第 N 期后提前还款
- 支持直接填写提前还款金额
- 支持按“提前还多少期”折算对应本金
- 对比“月供不变，缩短期限”和“期限不变，降低月供”
- 显示节省利息、净节省和关键时间线

## 本地运行

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:4173/
```
